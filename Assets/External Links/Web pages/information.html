Q1: A firewall is a device that resides on a network. It monitors traffic coming in and going out of the network as well as prevents certain types of traffic from entering the network

Q2: Usually, a URL beginning with HTTPS is secure as it stands for HyperText Transfer Protocol Secure. The difference is that HTTPS uses SSL (Secure Socket Layer) to encrypt its messages.

Q3: A brute force attack is classified as an access attack which generally means the user of harmful techniques to try and gain access into a system or network.

Q4: A zombie is a computer that has been infected with malicious code from an attacker that can be used remotely to carry out malicious tasks such as a DDoS (Distributed Denial of Service) attacks.

Q5: DoS stands for Denial of Service. What this does is prevents user from access a resource on a network due to it being constantly flooded with traffic from an attacker.

Q6: Malware includes different types of threats such as trojan horses, viruses, and worms. These malwares can really damage your system once your system gets infected. It is best to use anti malware or antivirus software to prevent this.

Q7: DHCP snooping is configured on switches in a network and acts like a firewall for between a DHCP server and other parts of the network. What its role is, is to monitor all untrusted ports on a switch and make sure no spoofed DHCP packets are getting to its destination

Q8: Clicking on links sent in emails can be very risky as it can lead to phishing. Phishing is when someone pretending to be a legitimate company sends an email and that contains a link that redirects a user to a mock website looking like the legitimate company website. The user will then be prompted to give their personal information one way or another to the fake website.
